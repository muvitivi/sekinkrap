Craftsman: Crafting & Building Add Server Add On

This add on will allow Craftsman CAB 2 to be used to add servers. By using this Add On, there will be a server menu that is not in the previous CAB Craftsman Menu.

Introduction

By using this Add On you can play / join the server.

Add Server Menu

You just have to install this lightweight Add On, and the server menu will automatically be available. This Add On will not change all the menu displays, will only change the play screen and add the Server menu.

  • Installation

    Download the add-on, Find files on Folder Craftsman CAB Add On, Rename .zip into .mcpack, open it, Select Open With Craftsman: Crafting & Building, then Craftsman CAB 2 should automatically install it for you. Select add on from setting. Done!
